Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je vysokoúrovňový, interpretovaný programovací jazyk, který se často používá k vývoji interaktivních webových stránek a webových aplikací.
  • Byl vytvořen v 90. letech a od té doby se stal klíčovým nástrojem ve vývoji webových technologií.
  • Syntaxe JavaScriptu je podobná jiným programovacím jazykům, jako je C, C++, a Java, což usnadňuje učení a používání.
  • JavaScript umožňuje vytvořit interaktivní prvky na webových stránkách, jako jsou formuláře, tlačítka, animace, záložky, otevírání a zavírání obsahů, dynamické grafy atd.
  • S JavaScriptem můžete měnit obsah stránky v reálném čase. Například můžete přidávat nebo odebírat prvky, měnit textové nebo obrazové obsahy a provádět různé úpravy stránky podle uživatelských akcí.
  • JavaScript umožňuje provádět asynchronní komunikaci s webovým serverem pomocí technologie AJAX (Asynchronous JavaScript and XML). Tím je možné načítat nebo odesílat data na server bez nutnosti znovunačítání celé stránky.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Choose meaningful and descriptive names for your variables, functions, and classes. This makes your code easier to understand. For example, use userName instead of u or calculateTotalPrice instead of calcTP.
  2. Adhere to a consistent coding style and formatting throughout your codebase. Common conventions include using camelCase for variable and function names, proper indentation, and a clear and consistent structure. Tools like ESLint can help enforce coding style rules.
  3. Use comments to explain the purpose and functionality of your code, especially for complex or non-obvious sections. This helps other developers (and your future self) understand your code.
  4. Minimize the use of global variables to prevent naming conflicts and unexpected behavior. Use local variables and limit the scope of your variables to the smallest necessary scope.
  5. Implement proper error handling to gracefully manage unexpected situations. Use try-catch blocks for exceptions, validate user inputs, and provide meaningful error messages. This improves the robustness of your code.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // s přiřazením prázdné (žádné) hodnoty
let cele = 5; // celé číslo (typ number)
let realne = 7.5; // reálné číslo (typ number)
let desetinne = 5e2; // zápis desetinného čísla pomocí exponentu
let text = "Ahoj"; // typ string
let prestupny = false; // proměnná typu boolean, nabývá pouze hodnot true nebo false

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, /=, *=, %=, ++, --, += (pro spojování řetězců)

Logické

&& (a zároveň), || (nebo), ! (negace), != (nerovná se), ? : (ternární operátor)

Porovnávací

== (rovná se), === (strikně se rovná, posuzuje i datový typ)
Pro porovnávání čísel:
< (menší), <= (menší nebo rovno), > (větší), >= (větší nebo rovno)
Větší a menší se také používá k porovnávání lexikografického pořadí

Bitové

&, |, ~, ^, <<, >>, >>>

Typové

typeof (typ proměnné), instanceof (), cosntructor

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

// Zobrazí se vyskakovací okno se zprávou Nezadal jsi správné heslo!
alert('Nezadal jsi správné heslo!');

prompt()

// Do proměnné password vloží heslo zadané uživatelem
let password = prompt('Zadej heslo:');

confirm()

// Zobrazí se vyskakovací okno, které se uživatele ptá, jestli chce přesměrovat na stránku seznam.cz
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

  1. Prohlížeč může vyskakovací okna blokovat.
  2. Stránky s vyskakovacími okny nevypadá důvěryhodně.
